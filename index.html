The Abstract Rewriting Machine (ARM) is a virtual machine which implements term rewriting for minimal term rewriting systems. Minimal term rewriting systems are left-linear term rewriting systems in which each rule takes on one of six forms: Each of these six forms is mapped (in ARM) to one or a few processor instructions on most contemporary micro processors.
